How to set conversions auto-pilot?

One-time setup. Ongoing conversions

How to turn on auto-conversion:

  • 1. Select currencies pair e.g. USD/EUR
  • 2. Schedule you due date e.g., 1st of each month
  • 3. Set the desire rate e.g., EUR/USD > 1.08
  • 4. Set a unique name e.g. “One-time EUR Purchase”

Takes 2 minutes • No integration needed to start
