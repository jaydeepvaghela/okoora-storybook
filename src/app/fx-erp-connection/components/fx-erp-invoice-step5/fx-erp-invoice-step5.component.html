Step 2: How to define your rules

  • 1. Select a category to protect ERP data - Protect invoices and/or bills.
  • 2. Select currencies - Protect cross-border transactions
  • 3. Set amount range - Transactions in this range (e.g., $1,000–$25,000) will be protected
  • 4. Select due term - Only those transactions (e.g., Net 30/60/90) will be protected
  • 5. Exclude contacts - Their transactions won’t be auto-protected
